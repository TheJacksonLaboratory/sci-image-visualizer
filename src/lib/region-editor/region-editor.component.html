The Regions tab lists every region drawn on the currently displayed image. Regions are tracked per image: switching to another image hides these and shows that image's regions; switching back restores them. Edits in the table apply live to the diagram — there is no save or cancel.

Creating regions

Use the diagram toolbar tools to draw new regions: rectangle, polyline, freeform path, or wand. Anything you draw appears immediately in the table, with the class name "legend" by default.

Editing

  • Click the "" toggle to expand a row. For rectangles you can click the x, y, width, or height cells to edit them inline. For polygons, click any X or Y coordinate cell to edit it directly.
  • For polygons, click the button on the right-hand side of the coordinates paginator to append a new point coordinate.
  • Click the Class cell to rename the region's class label.
  • Long coordinate strings are truncated with "…"; hover the cell to see the full value.
  • Use "x512" to round all rectangle widths/heights to the nearest multiple of 512.

Selecting rows

  • Click a row's Class or Coordinates cell to select it. The corresponding region is highlighted in the diagram (Plotly shows its edit handles; the OpenSeadragon image view highlights it and shows its handles in select mode).
  • Ctrl/Cmd+click to add or remove a row from the selection. Shift+click to extend the selection in a range.
  • Selecting a region in the diagram highlights the matching row in the table.
  • Clicks on the chevron, the trash icon, or the active class-edit input do not change the selection — that's standard PrimeNG behaviour.

Bulk actions on the selected rows

  • Edit the label on every selected row at once.
  • Delete every selected region — both from the table and the diagram.
  • Delete every region on this image.

Classification colours

  • Edit the colour of every classification present in the table from one dialog — no selection required. Each class gets a colour swatch; applying recolours all regions of that class and remembers the colour for future regions with the same label.

Per-row actions

  • at the end of a row deletes that one region.

Display options

  • Toggle showing class labels on every region in the diagram (slower with many regions).
  • The colour swatch sets the fill colour applied to the active region in the diagram.

Import / export

  • Import regions from a GeoJSON file.
  • Export the current image's regions to a GeoJSON file. (The per-image cache is in-memory only — exporting is how you persist regions across sessions.)
