SAM models

All are promptable Segment Anything models finetuned for biomedical images. They differ in the image encoder (the heavy once-per-image step) and the training domain — box and point prompting behave the same.

micro-sam ViT-T (default)
Light-microscopy finetuned. TinyViT encoder, ~14 MB, very fast, runs on WASM. Good mask quality — best for quick interactive work and many/large objects.
micro-sam ViT-B
Light-microscopy finetuned. ViT-Base encoder, ~172 MB, slower, GPU-accelerated (WebGPU). Highest accuracy — when ViT-T misses subtle or overlapping boundaries.
patho-sam ViT-B
Histopathology finetuned (H&E / stained tissue). ViT-Base encoder, GPU-accelerated. Use it for pathology slides — it segments tissue/nuclei better than the microscopy models.
patho-sam ViT-B (int8)
Same model, int8-quantized — ~100 MB vs ~180 MB (smaller download). Runs on WASM (slower encode than the fp16 version); masks are near-identical. Pick it on slow connections.
The choice applies to both the box and point tools. Each tool loads its own encoder, so the first click after switching re-encodes once.

Click a file in the left-hand tree to display it. By default it opens in the Image view — a natively tiled, deeply zoomable raster (OpenSeadragon) with a navigator minimap and a physical-units scale bar. Click the image to zoom toward the clicked point; scroll to zoom and drag to pan. When a stack is open, a slice slider (Image view) or image-mode toggle (other views) appears. Regions can also be edited from the right-panel "Regions" tab. Drag the handle to detach this toolbar into a floating window.

Plot-type dropdown

The plot-type dropdown switches how the file is visualized: Image, Contour, Scatter (regions), Surface (3D), Scatter 3D, and Isosurface (3D). Scalar and 3D types require a grayscale image (the 3D volume types also require a z-stack). The non-image views render with Plotly and support "real zooming" (a downscaled overview that re-fetches higher resolution as you zoom in).

  • "Isosurface band" slider (Isosurface): sets the intensity band the surfaces are drawn for, mapped onto the volume's real intensity range.
  • "3D camera" (Surface / Scatter 3D / Isosurface): zoom , pan , orbit , turntable , and reset camera ; scroll to zoom the 3D scene.

Image manipulation tools

  • "Intensity profile lines" (work in progress — not yet stable): adds a coloured line ROI; a floating inset chart plots intensity along each line, updates live as you drag it, and re-samples at the current zoom resolution. Each added line gets its own colour; select a line and use the trash tool to remove it (and its trace).
  • "Channels & Histogram" : opens a floating dialog for brightness/contrast, gamma, and per-channel display, plus the colormap / LUT controls (grayscale only) — a reverse toggle and a colormap dropdown (default: inverted greys).
  • "Smoothen" (Image view only): the image opens showing raw pixels (nearest-neighbour), so zooming in renders crisp blocks for pixel inspection. Toggle this on for bilinear smoothing; the icon turns white while smoothing is active.

Download & view tools

  • "Download" : download the current image as a PNG.
  • "Fit to view" / autoscale : resets the image to its original dimensions. In stack mode it also resets to single-image mode.

Zoom tools

  • "Zoom box" (b): drag a box to select an area to zoom into.
  • "Pan" (p): drag to pan the image.
  • "Zoom in" (+): zoom in one step.
  • "Zoom out" (-): zoom out one step, stopping at the original image dimensions.

Region tools

By default no tool is selected (all greyed-out); selecting a tool highlights it. The vertex-editing tools (polygon, add/delete vertex, move, Bézier conversions) are available in the Image view only. On a donut (a region with a hole), the inner-ring vertices are editable too — drag, add, and delete them just like the outer ring.

  • "Selection" (s): a neutral mode that turns off whatever drawing tool was active (wand, vertex eraser, rectangle, etc.). Use it to step out of editing without picking another tool.
  • "Rectangular region" (r): click and drag to create a rectangular region.
  • "Polyline" (l): draw an open (non-closed) polyline by clicking and dragging. Polylines can be edited and exported to GeoJSON as a LineString feature.
  • "Freeform region" (f): click and drag to draw a closed freeform polygon.
  • "Brush" : click and drag to paint a region with a circular brush — the painted stroke is the region. A Size slider (brush diameter, in image pixels) appears in the toolbar while the brush is active. Hold Shift while painting to erase (subtract) from a region instead. Painting over an existing region extends it; brushing across two regions merges them. Painting a closed ring (or shift-erasing an interior pocket) leaves the enclosed area empty — a donut (Image view).
  • "Polygon" : click to place each vertex; click the first point to close the polygon.
  • "Add vertex" : click an edge of the selected polygon to insert a new vertex.
  • "Delete vertex" : click a vertex of the selected polygon to remove it.
  • "Move region" : drag to move the selected region.
  • "To Bézier" / "To polygon" : convert the selected region to a smooth Bézier curve, or back to a straight-edged polygon.
  • "Wand" (w): click to grow a region from pixels with similar values around the click point. Hold Ctrl/Cmd for an exact-match flood fill; drag to keep extending the selection while moving; hold Shift while clicking or dragging on an existing region to erase pixels from it. Use the Sens. slider that appears to make the wand stricter (higher = smaller selection) or looser. Subsequent clicks keep extending the same region; clicking inside an existing region adopts it, and growing into another region merges them.
  • "Vertex eraser" (e): click or click-and-drag to remove polygon vertices that fall inside the cursor circle from any region, including a region's inner rings (donut holes). Use the Radius slider to set the eraser size in image pixels. Polygons reduced below 3 vertices (or polylines below 2) are deleted entirely; a hole reduced below 3 vertices is removed.
  • "Undo" (Ctrl/Cmd+Z): reverts the last region action (draw, delete, edit, wand/brush stroke, segmentation, …). Up to the last 10 actions can be undone; the button greys out when there's nothing left to undo.
  • "Redo" (Ctrl/Cmd+Shift+Z or Ctrl/Cmd+Y): re-applies the most recently undone action. Performing a new region action clears what can be redone.
  • "Delete region" (Del): deletes the selected region. This can also be done from the Regions tab.

Region actions (right-click the image)

Right-click a region — or the current multi-selection — to open the "Selected region(s)" group at the top of the context menu. These act immediately on the selection (they're not drawing modes) and are fully undo/redo-able:

  • "Select all regions" : select every region on the image (also in the Regions tab toolbar).
  • "Merge / group": combine 2+ selected regions into one. If the pieces don't touch, the result is a single multi-part region (sparse "bits").
  • "Ungroup": split a multi-part region back into one region per piece.
  • "Inverse": replace the selection with everything outside it — the image rectangle with the selected regions punched out as holes. Works at any image size.
  • "Simplify": reduce a region's vertex count (Douglas–Peucker) by an altitude threshold in pixels — pick Light (1 px), Medium (3 px), Strong (8 px), or Custom… for a slider. Applies to the exterior and any holes (and every part of a multi-part region).
  • The Bézier conversions and Delete also live in this group (the right-click equivalents of the toolbar / vertex tools above).

Segmentation tools (SAM & cellpose)

AI-assisted segmentation that runs entirely in the browser. The box and point tools use a promptable micro-sam model; the cellpose tool auto-detects cells. Generated regions inherit the color of the rectangle they came from. The first run downloads the model (a progress toast is shown); it's then cached.

  • "Segment" (box prompt): draw one or more Rectangular regions around the objects of interest, then click Segment. Each rectangle is replaced by the SAM mask of the object inside it.
  • "Point prompts" : click directly on an object to segment it as a new region. Each click starts a fresh object — clicking another object won't grow the previous one. Shift/Alt-click adds an exclude point that refines the current object; Esc undoes it, Enter commits it.
  • "Model picker" : the dropdown attached to the Segment button chooses the SAM model — micro-sam ViT-T (default; small, fast) or micro-sam ViT-B (larger, highest accuracy) for light microscopy, or patho-sam ViT-B for histopathology (H&E / stained tissue). The choice applies to both the box and point tools. The info button next to it explains the trade-off.
  • "Cellpose" : draw rectangles, then click Cellpose to automatically segment every cell inside each rectangle (client-side cellpose-SAM) — one region per detected cell. No clicking on individual cells needed.
