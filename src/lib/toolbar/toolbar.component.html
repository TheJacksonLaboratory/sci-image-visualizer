Click a file in the left-hand tree to display it. By default it opens in the Image view — a natively tiled, deeply zoomable raster (OpenSeadragon) with a navigator minimap and a physical-units scale bar. Click the image to zoom toward the clicked point; scroll to zoom and drag to pan.

The plot-type dropdown switches how the file is visualized: Image, Heatmap, Contour, Scatter (regions), Surface (3D), Scatter 3D, and Isosurface (3D). Scalar and 3D types require a grayscale image (the 3D volume types also require a z-stack). The non-image views render with Plotly and support "real zooming" (a downscaled overview that re-fetches higher resolution as you zoom in).

The colormap / LUT controls (grayscale images only) include a "reverse" toggle and a dropdown of colormaps (default: inverted greys). When a stack is open, the image mode and z-slice can be changed. Regions are edited in the right-panel "Regions" tab. Drag the handle to detach this toolbar into a floating window.

Plot-type controls

  • "Intensity profile lines" (Heatmap & Image): adds a coloured line ROI; a floating inset chart plots intensity along each line, updates live as you drag the line, and re-samples at the current zoom resolution. Each added line gets its own colour; select a line and use the trash tool to remove it (and its trace).
  • "Isosurface band" slider (Isosurface): sets the intensity band the surfaces are drawn for, mapped onto the volume's real intensity range.
  • "3D camera" (Surface / Scatter 3D / Isosurface): zoom , pan , orbit , turntable , and reset camera ; scroll to zoom the 3D scene.

The other controls include the tool bar, where by default not tool is selected (all greyed-out), and which allows

  • "image download" : download the image as a 'png'.
  • "autoscale" : resets the image to its original dimension. If in stack mode, it will reset the image mode to single image mode.
  • "zoom drag" : activate zoom mode to perform a zoom selection on the image. If in single-image mode, or any image that is not a stack, the mouse selection will trigger a high-def zoom, creating a cache of the image the first time (longer loading); then the next zooming events will be faster than the first one.
  • "zoom box" : activate zoom box mode to select an area to zoom into.
  • "panning" : activate panning image mode to perform a panning selection; dragging and releasing the mouse on a zoomed image will trigger a new zoom event on the new area of the image selected.
  • "zoom-in" : triggers the zoom-in event where the zoom selection is automatically set.
  • "zoom-out" : triggers the zoom-out event where the zoom selection is automatically set. The last zoom selection will stop at the original image dimensions.
  • "freeform region creation" : activates the freeform region mode which enables creation of a closed freeform polygon using the mouse, by click and dragging the left-mouse button.
  • "polyline drawing" : activates polyline mode which enables drawing of an open (non-closed) polyline using the mouse, by clicking and dragging the left-mouse button. Polylines can be edited and exported to GeoJSON as a LineString feature.
  • "rectangular region creation" : activates the rectangular region mode which enables creation of a rectangular selection using the mouse, by clicking and dragging the left-mouse button.
  • "wand" : click on the image to grow a region from pixels with similar values around the click point. Hold Ctrl (or Cmd) for an exact-match flood fill. Drag the mouse to keep extending the selection while moving. Hold Shift while clicking or dragging on an existing region to erase pixels from it. Use the Sens. slider that appears in the toolbar to make the wand stricter (higher = smaller selection) or looser. Subsequent clicks keep extending the same region; clicking inside an existing region adopts it, and growing into another region merges them.
  • "vertex eraser" : click or click-and-drag to remove polygon vertices that fall inside the cursor circle from any region. Use the Radius slider to set the eraser size in image pixels. Polygons reduced below 3 vertices (or polylines below 2) are deleted entirely.
  • "selection" : a neutral mode that turns off whatever drawing tool was active (wand, vertex eraser, rectangle, etc.). Use it when you want to step out of editing without picking another tool.
  • "active region deletion" : deletes the selected region. This can also be done from the Regions tab.
